# DOC_TEMPLATE — PICO UI 文档输出模板

> 版本: 1.0.0 | 创建: 2026-03-24 | 状态: 生效中

提供组件文档、Plan、Tasks、Implement 的标准 Markdown 模板。

详细模板内容见仓库完整文件。

## 模板列表

1. **组件文档**: 概述 / 快速开始 / API 参考 / 组件结构 / 主题定制 / 已知限制
2. **Plan**: 需求摘要 / 影响分析 / 技术方案 / 风险评估 / 验收标准 / 估时
3. **Tasks**: 任务列表 / 依赖 DAG / 状态跟踪
4. **Implement**: 变更清单 / 自检结果 / 测试结果 / 遗留问题
