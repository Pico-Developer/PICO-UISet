# WORKFLOW — PICO UI Spec Coding 工作流

> 版本: 1.0.0 | 创建: 2026-03-24 | 状态: 生效中

## 8 阶段工作流

1. 需求输入 - 接收需求, Figma 同步
2. Plan - 技术方案, 影响分析, 风险评估
3. Tasks - 任务拆解, 依赖 DAG
4. Implement - 代码实现, 自检验证
5. 测试 - 自动化测试, 视觉回归
6. 修复 - Bug 修复, 回归测试
7. 文档输出 - API 文档, 使用指南
8. 交付确认 - 人工 Review, 合入主分支

| 阶段 | 检查点 | 阻断? |
